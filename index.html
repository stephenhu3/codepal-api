CodePal

CodePal is integrated with StackOverflow so you can say goodbye to the clutter of managing multiple tabs. Your favorite forum for searching, discussing, and debugging your issues is accessible right on the cohesive interface.

CodePal is also integrated with YouTube so that you can easily keep up with programming tutorials and learn new technologies with ease by running both the code editor and video within the same window.

CodePal has a built-in editor that allows you to write and run your code in various languages within the browser. Now, you won't have to go through the trouble of launching a full-fledged IDE for testing your code snippets.
